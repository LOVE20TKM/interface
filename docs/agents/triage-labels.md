# Triage Labels

| Canonical role    | Local status      | Meaning                       |
| ----------------- | ----------------- | ----------------------------- |
| `needs-triage`    | `needs-triage`    | Waiting for maintainer triage |
| `needs-info`      | `needs-info`      | Waiting for more information  |
| `ready-for-agent` | `ready-for-agent` | Ready for an AI agent         |
| `ready-for-human` | `ready-for-human` | Requires human implementation |
| `wontfix`         | `wontfix`         | Will not be actioned          |

When a skill mentions a canonical role, use its corresponding local status.
